# Unity 2D Platformer
![Демонстрация](Assets/demostration.gif)

## Описание
Игрок управляет персонажем, который перемещается по уровню, прыгает по платформам, сражается с врагами и собирает монеты. 

## Управление
| Действие          | Клавиша |
|-------------------|---------|
| Движение влево    |      ←/A|
| Движение вправо   |      →/D|
| Прыжок            |    Space|
| Атака             |       С |

## Особенности
- Управление персонажем (движение, прыжок, атака)
- Враги с патрулированием и уязвимостью
- Система здоровья и урона
- Сбор предметов и счётчик очков
- UI: здоровье, счёт, главное меню с выбором уровней, меню перезапуска уровня
- Звуковые эффекты: прыжок, сбор, урон

## Установка зависимостей
Для корректной работы проекта необходимо установить следующие пакеты через Unity Package Manager:

1. Откройте Unity → Window → Package Manager.
2. Нажмите "Add package from git URL…" и добавьте:

   - UniTask: `https://github.com/Cysharp/UniTask.git?path=src/UniTask`
   - DOTween: `http://dotween.demigiant.com/download.php` (или через Asset Store)
   - Zenject: `https://github.com/modesttree/Zenject.git`

3. После установки всех пакетов, проект будет готов к открытию и запуску.
